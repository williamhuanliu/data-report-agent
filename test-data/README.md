# 测试数据说明

用于多文件上传与报告生成的测试 CSV，均为 UTF-8 编码。**歌曲、歌手、厂牌通过 ID 关联**，便于做综合分析。

## 关联关系

```
厂牌 (厂牌ID) ←── 歌手 (厂牌ID) ←── 歌曲 (歌手ID) ←── 播放量 (歌曲ID)
```

| 文件 | 说明 | 外键 / 关联 |
|------|------|-------------|
| **厂牌.csv** | 厂牌ID、厂牌名、成立年份、地区、旗下艺人数量 | 主键：厂牌ID |
| **歌手.csv** | 歌手ID、歌手名、厂牌ID、出道年份、国籍 | 主键：歌手ID；外键：厂牌ID → 厂牌 |
| **歌曲.csv** | 歌曲ID、歌曲名、歌手ID、发行日期、时长、风格 | 主键：歌曲ID；外键：歌手ID → 歌手 |
| **播放量.csv** | 日期、歌曲ID、播放量(万)、排名 | 外键：歌曲ID → 歌曲 |

- **歌曲.歌手ID** → 对应 **歌手.歌手ID**
- **歌手.厂牌ID** → 对应 **厂牌.厂牌ID**
- **播放量.歌曲ID** → 对应 **歌曲.歌曲ID**

在仪表盘「新建报告」中可一次上传以上多个文件，AI 将按歌曲→歌手→厂牌与播放量做关联分析并生成报告。
