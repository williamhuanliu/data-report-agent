# 测试数据说明

用于多文件上传与报告生成的测试 CSV，均为 UTF-8 编码。**歌曲、歌手、厂牌通过 ID 关联**，便于做综合分析。数据量较丰富，覆盖多厂牌、多歌手、多风格与全年播放趋势。

## 关联关系

```
厂牌 (厂牌ID) ←── 歌手 (厂牌ID) ←── 歌曲 (歌手ID) ←── 播放量 (歌曲ID)
```

| 文件 | 说明 | 数据量 | 外键 / 关联 |
|------|------|--------|-------------|
| **厂牌.csv** | 厂牌ID、厂牌名、成立年份、地区、旗下艺人数量、**主营业务、注册资本万、是否上市、合作平台数** | 10 条 | 主键：厂牌ID |
| **歌手.csv** | 歌手ID、歌手名、厂牌ID、出道年份、国籍、**性别、出生年份、代表作品数、微博粉丝万、活跃度** | 25 条 | 主键：歌手ID；外键：厂牌ID → 厂牌 |
| **歌曲.csv** | 歌曲ID、歌曲名、歌手ID、发行日期、时长、风格、**语言、专辑名、是否主打、收藏数万** | 85 条 | 主键：歌曲ID；外键：歌手ID → 歌手 |
| **播放量.csv** | 日期、歌曲ID、播放量(万)、排名、**环比增长率、分享次数万、评论数万** | 240 条（2024 年 1–12 月，每月 Top 20） | 外键：歌曲ID → 歌曲 |

- **歌曲.歌手ID** → 对应 **歌手.歌手ID**
- **歌手.厂牌ID** → 对应 **厂牌.厂牌ID**
- **播放量.歌曲ID** → 对应 **歌曲.歌曲ID**

## 数据特点

- **厂牌**：10 家，含主营业务（唱片/演出/版权/经纪等）、注册资本、是否上市、合作平台数；地区含台北、北京、上海、新加坡、香港等。
- **歌手**：25 位，含性别、出生年份、代表作品数、微博粉丝万、活跃度（高/中/低）；覆盖各厂牌，国籍含中国、新加坡、马来西亚、美国等。
- **歌曲**：85 首，含语言（国语/粤语）、专辑名、是否主打、收藏数万；风格含流行、民谣、中国风、说唱、摇滚、电子、R&B、抒情等。
- **播放量**：2024 年全年 12 个月，每月 Top 20；含环比增长率（%）、分享次数万、评论数万，便于做传播与互动分析。

在仪表盘「新建报告」中可一次上传以上多个文件，AI 将按歌曲→歌手→厂牌与播放量做关联分析并生成报告。
